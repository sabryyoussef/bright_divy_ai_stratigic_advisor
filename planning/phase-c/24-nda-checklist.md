# NDA — execution checklist (Step 24)

**Not legal advice.** Use counsel-approved paper for Qatar / cross-border as needed.

## Before signature

- [ ] Confirm **counterparty** legal name and signatory (NexSolve or affiliate).
- [ ] Choose template: **mutual** vs one-way; jurisdiction and governing law agreed.
- [ ] Define **confidential information**: FSD, data samples, API docs, test sets, call notes.
- [ ] **Duration** and **return/destruction** of materials post-RFQ.
- [ ] **Residuals** and **independent development** clause understood by both sides.
- [ ] If **subcontractors** or cloud used later: they are bound or excluded per counsel.

## Execution

- [ ] **Sign** (wet or qualified e-sign per policy).
- [ ] Store **executed PDF** in secure location (not public Git). Internal reference: `[DATE-NDA-NexSolve]`.
- [ ] Distribute to: [names].

## After NDA

- [ ] **Receive FSD** and supporting artefacts under the agreed terms (see `25-fsd-intake-checklist.md`).
- [ ] Update CRM / opportunity record: NDA in place.

**Master plan:** Mark Step 24 as finished only when an executed NDA is on file.
